# Claude Code Remote - 内部试用发布

> 从任意设备远程控制PC上的Claude Code

## 产品简介

**Claude Code Remote** 让你从任意设备（手机、平板、电脑）远程控制PC上运行的Claude Code。

**一句话**：躺在床上用手机写代码，Claude在PC上帮你干活。

---

## 核心功能

| 功能 | 说明 |
|-----|------|
| :globe_with_meridians: 远程访问 | 随时随地访问你的开发环境 |
| :speech_balloon: 实时会话 | 双向实时通信，响应即时 |
| :file_folder: 文件浏览 | 远程浏览项目文件结构 |
| :twisted_rightwards_arrows: Git操作 | 远程执行Git命令 |
| :chart_with_upwards_trend: Token统计 | 实时查看API消耗 |

---

## 适用场景

- 没有Claude订阅，想用官方remote功能
- 出差/通勤时想继续coding
- 管理多台PC上的多个工程

---

## 技术架构

```
┌─────────────┐          ┌──────────────┐          ┌─────────────┐
│ 客户端      │◄────────►│   云服务器    │◄────────►│ PC守护进程  │
│ (Web/PWA)   │ Socket.io│ (Express)    │ Socket.io │ (Agent)     │
└─────────────┘   +JWT   └──────────────┘   +JWT   └─────────────┘
                               │                        │
                               ▼                        ▼
                        ┌──────────────┐         ┌─────────────┐
                        │   SQLite     │         │ Claude Code │
                        │ (Prisma ORM) │         │   进程      │
                        └──────────────┘         └─────────────┘
```
