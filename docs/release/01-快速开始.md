# 快速开始

5分钟完成部署，开始远程开发。

---

## 前置要求

| 组件 | 最低要求 |
|-----|---------|
| PC系统 | macOS / Linux / Windows (WSL2) |
| Node.js | 18.x+ |
| 内存 | 2GB+ |
| 网络 | 能访问云服务器 |

> :bulb: 开始前，请确保已获取 **服务器地址** 和 **JWT Token**

---

## 三步上手

### Step 1: 下载并运行安装脚本

```bash
# 下载并执行安装脚本
curl -fsSL https://your-server/cc-remote/scripts/install.sh | bash
```

脚本会自动：
- :white_check_mark: 检测系统环境
- :white_check_mark: 安装依赖（Node.js、pnpm）
- :white_check_mark: 下载最新版本
- :white_check_mark: 配置服务

<details>
<summary>手动安装（可选）</summary>

```bash
# 1. 克隆仓库
git clone https://github.com/your-org/cc-remote.git
cd cc-remote

# 2. 安装依赖
pnpm install

# 3. 构建项目
pnpm build
```

</details>

---

### Step 2: 绑定你的PC

```bash
cd cc-remote

# 绑定Token
pnpm agent:bind --token <你的JWT-Token>

# 启动Agent
pnpm agent:start
```

看到 :white_check_mark: `Agent已连接` 即表示成功。

---

### Step 3: 打开Web界面

1. 访问 **https://your-server.com**
2. 使用你的账号登录
3. 在PC列表中看到你的电脑

:tada: **恭喜！** 现在可以用手机远程操作Claude Code了。

---

## 常用命令

| 命令 | 说明 |
|------|------|
| `pnpm agent:start` | 启动Agent |
| `pnpm agent:stop` | 停止Agent |
| `pnpm agent:status` | 查看状态 |
| `pnpm agent:restart` | 重启Agent |
